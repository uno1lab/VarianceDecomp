# VarianceDecomp
Decomposing variance of the distribution of the cluster-specific proportions for binary outcome data

- fun-regional-var.R : the main function for the analysis
- run-example.R : The sample code to run the main function

Please see the comments in the program code for the usage. 

Reference: Uno H, Tramontano AC, Punglia RS, Hassett MJ. Decomposing Variations on Cluster Level for Binary Outcomes in Application to Cancer Care Disparity Studies. Health Serv Res. 2025 Mar 18:e14599. doi: 10.1111/1475-6773.14599. Epub ahead of print. PMID: 40102175.
